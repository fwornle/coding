| T-32-04 | T (Tampering) | .coding/session-state.json | accept | Local file in gitignored directory, same trust as project files. No remote write path. |

| T-32-06 | D (DoS) | execSync in write-session-state.js | mitigate | Set timeout: 3000ms on all execSync calls. Fail-open on timeout. |
