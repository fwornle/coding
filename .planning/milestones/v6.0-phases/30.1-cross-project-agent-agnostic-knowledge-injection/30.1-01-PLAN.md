Add a new method `_applyContextBoost(results, context)`:
- `context.project` (string): If a result's payload contains the project name (case-insensitive match in `payload.project`, `payload.source`, or `payload.text`), boost its `rrfScore` by 15% (multiply by 1.15)
- `context.cwd` (string): If a result's payload text contains a path segment from cwd (last 2 path components, e.g. "Agentic/coding"), boost by 10% (multiply by 1.10)
- `context.recent_files` (string[]): If a result's payload text mentions any of the recent file basenames, boost by 20% (multiply by 1.20)
- Boosts are cumulative (a result matching all three gets 1.15 * 1.10 * 1.20 = ~1.518x)
- If context is null or empty object, skip entirely (backward compatible)
