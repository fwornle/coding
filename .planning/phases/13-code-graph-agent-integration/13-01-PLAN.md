- Export `CgrQueryCache` class:
  - `private cache = new Map<string, CgrComponentData>()`
  - `private cgrAgent: CodeGraphAgent`
  - `private available = true`
  - `private refreshPromise: Promise<void> | null = null`
  - Constructor takes `repositoryPath: string`, creates `CodeGraphAgent`
  - `async refreshIndex(timeoutMs: number = 30000): Promise<void>` — calls `cgrAgent.hasExistingIndex()`, then `cgrAgent.indexRepository({ incremental: hasIndex })`. On timeout, logs warning and continues. On error, sets `available = false`. Stores as `refreshPromise` so it can be awaited later.
  - `async ensureReady(): Promise<void>` — awaits `refreshPromise` if pending
  - `isAvailable(): boolean`
  - `async queryComponentEntities(componentName: string, keywords: string[]): Promise<CodeEntity[]>` — checks cache first (key = componentName). If miss, runs Cypher query scoped to component file paths (use `MATCH (f:File)-[:DEFINES]->(e) WHERE toLower(f.file_path) CONTAINS toLower(...)` pattern). Sanitize componentName with `sanitizeCypher()` helper (replace single quotes with escaped quotes). Cache result. Return entities.
  - `async queryEntityDetails(entityName: string, componentFiles: string[]): Promise<CgrEntityDetails>` — Cypher query for entity signatures + callees + imports. NOT cached (per-entity, not per-component).
  - `async queryCallGraph(entityName: string, depth: number = 2): Promise<CgrCallGraphResult>` — Cypher query for call graph traversal to specified depth. NOT cached.
  - `getStats(): { queriesMade: number, cacheHits: number, available: boolean }` — for dashboard stats
  - Private `sanitizeCypher(input: string): string` helper — escapes single quotes and backslashes
  - All query methods: wrap in try/catch, return empty results on failure (never throw per locked decision), log warnings with `[CgrQueryCache]` prefix
