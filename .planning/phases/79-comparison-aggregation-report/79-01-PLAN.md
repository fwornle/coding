<objective>
Persist the OBJECTIVE per-cell test-gate outcome as a discrete queryable field (`gate_passed: true|false|null`) on the Score entity at score time. This is CMP-01's enabler (D-04a part A): the evidence harness ALREADY computes the objective test result at close time, but today it is only consumed to derive subjective rubric dims — it is never persisted as a queryable gate. Phase 79's aggregator (Plan 02) needs this field to separate gate-passers from failed/ungated runs.

Purpose: Without a persisted gate field, re-deriving the gate at compare time is infeasible — the agent's edits lived in an ephemeral sandbox worktree destroyed after the run (D-04). This is a small additive change: the harness result is already in-hand.
Output: `gateFromEvidence()` helper, the threading in measurement-stop, the persisted `gate_passed` field on Score, and a unit test.
</objective>
