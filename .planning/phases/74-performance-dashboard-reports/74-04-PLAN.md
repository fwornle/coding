| dashboard server.js → vkb-server (:8080) | Same-origin reverse proxy to the experiment API |
| vkb-server handler → experiment LevelDB / token-usage.db | Untrusted task_id/report_id reach store + SQL |

| T-74-04-01 | Tampering | SQL injection via task_id in handleTimeline | mitigate | timeline-read.mjs binds task_id as `?` (Plan 02); handler passes req.params.taskId straight to the bound-param reader, never builds SQL |
