<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| origin span metadata → synthesized spec | fork reads a completed Run's prompt/snapshot; feeds goal_sentence/snapshot_id into a spec that drives headless spawn |
| avenue cell env → agent child | env-axis + injection flag + task_id binding cross into the spawned agent |
| avenue span writes → MAIN .data | measurement rows from a branch context land in the shared main store |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-87-03-01 | Tampering (cmd injection) | goal_sentence / branch name in synthesized spec | mitigate | goal_sentence carried as a data field (never shell); branch name derived via sanitizeTaskId; test_command shell-safety already enforced by SHELL_META_RE in validateCells (spec unchanged) |
| T-87-03-02 | Spoofing / mis-attribution | avenue task_id collision with origin | mitigate | composeTaskId guarantees a unique task_id per avenue; the AVN-06 test asserts disjoint aggregation (Pitfall 2) |
| T-87-03-03 | Repudiation / data loss | span isolated into sandbox → 0-token avenue | mitigate | span always opens with spanEnv → MAIN (:491); the AVN-06 test encodes the sandbox-span failure as a caught negative (Pitfall 1) |
| T-87-03-04 | Elevation | autonomous agent editing outside sandbox | mitigate | agent cwd = avenue worktree (isolated); only measurement (not code) writes to MAIN; commit-on-close is scoped to the avenue branch |
| T-87-03-05 | DoS / mis-attribution | concurrent avenues | mitigate | sequential await-each loop preserved (Pitfall 5); Plan 85 D-02 409 span-slot guard upstream |
| T-87-03-SC | Tampering | npm/pip/cargo installs | mitigate | ZERO installs (RESEARCH: no Package Legitimacy Audit needed); `git diff --stat package.json` gate is empty |
