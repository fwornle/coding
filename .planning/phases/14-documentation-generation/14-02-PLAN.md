   ```typescript
   const GENERIC_ENTITY_NAMES = new Set([

   ]);
   ```

2. **Add `PASCAL_CASE_REGEX`** constant: `/^[A-Z][a-zA-Z0-9]*$/`

3. **Create `validateEntityForPersistence()` private method** returning `{ valid: boolean; reasons: string[] }`:
   - Accepts `(entity: SharedMemoryEntity, allEntityNames: Set<string>)`
   - Runs four constraint rules in sequence, collecting ALL failure reasons (not short-circuiting):

   **Rule 1 - Entity naming:**
   - Check entity.name matches PASCAL_CASE_REGEX
   - Check entity.name is NOT in GENERIC_ENTITY_NAMES
   - Check entity.name length >= 3 and <= 80
   - Failure reasons: "Name '{name}' is not PascalCase", "Name '{name}' is generic/blocklisted", "Name '{name}' invalid length"

   **Rule 2 - Observation count:**
   - Require 3+ observations (reuse existing logic from validateEntityQuality)
   - Failure reason: "Insufficient observations ({count}/3 required)"

   **Rule 3 - Content quality:**
   - At least one observation must be multi-sentence (contains 2+ periods or semicolons)
   - Not ALL observations generic (reuse existing isGenericObservation)
   - At least one observation should reference code (contains file path pattern like `/` or `.ts`/`.js` or PascalCase identifier)
   - Failure reasons: "All observations are single-sentence stubs", "All observations are generic", "No code-grounded observations"

   **Rule 4 - Hierarchy integrity:**
   - hierarchyLevel must be defined and in range [0, 3]
   - If parentEntityName is set, it must exist in allEntityNames (or be the entity itself for L0)
   - hierarchyPath (if set) depth should match level (count of `/` separators + 1)
   - L0 entities should NOT have parentEntityName
   - Failure reasons: "Invalid hierarchy level ({level})", "Parent '{parent}' not found in entity set", "HierarchyPath depth mismatch"

4. **Replace existing validation in `persistWaveResult()`**:
   - Remove the two separate filter passes (structural + quality)
   - Build `allEntityNames` Set from all sharedMemoryEntities
   - Single filter pass: `sharedMemoryEntities.filter(e => { const result = this.validateEntityForPersistence(e, allEntityNames); ... })`
   - Log each rejection: `log('[WaveController] Constraint rejection: ${e.name} - ${result.reasons.join('; ')}', 'warning')`
   - Log summary: `log('[WaveController] Constraint gate: ${passed}/${total} entities passed', 'info')`

5. **Remove `validateEntityQuality()`** method (absorbed into new gate). Keep `isGenericObservation()` as it's reused.
