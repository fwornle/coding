<objective>
Refactor `ObservationWriter.writeInsight` (OW.js:1284) to return `{legacyId, mintedId}` instead of just the legacyId string (`row.id`). The mintedId is the freshly-minted km-core entity id, already in scope as the return of the internal `kmStore.putEntity` call at OW.js:1309. Surfacing it to callers closes the D-03 root cause: the consolidator's `_pushInsightToKG` at OC.js:660-661 currently re-derives the mintedId via a post-write `findByLegacyId({system:'A', id: entry.topic})` lookup that races against km-core's in-memory hydration — on a ~1-in-100 rate observed 2026-06-15 the lookup returns null, silently disabling the entire `has_insight` block at OC.js:679-705 and producing an orphan Insight.

Purpose: Eliminate the race at the writer (where the mintedId is known) instead of leaving the consumer to re-derive it. This is a writer-side signature change ONLY — the consumer refactor lands in Plan 59-03 once this plan's contract ships.

Output: New return shape `{legacyId: row.id, mintedId}`; updated JSDoc; three unit tests proving the contract.
</objective>

```
async writeInsight(row, options = {}) {
  if (!row || typeof row !== 'object') throw new Error('[ObservationWriter] writeInsight: row required');
  if (!row.id)                          throw new Error('[ObservationWriter] writeInsight: row.id required');
  const mentionsTargetIds = Array.isArray(options?.mentionsTargetIds) ? options.mentionsTargetIds : [];
  const kmStore = await this._ensureKmStore();
  const ts = row.created_at || new Date().toISOString();
  try {
    const entity = legacyInsightToEntity(row, this._runId, ts);
    if (!entity.ontologyClass) entity.ontologyClass = 'Detail';
    entity.metadata = { ...entity.metadata, source: entity.metadata?.source ?? 'auto' };
    const mintedId = await kmStore.putEntity(entity, { skipOntologyCheck: true });
    await this._emitMentionsEdges(kmStore, mintedId, mentionsTargetIds);
    await this._anchorEntity(kmStore, mintedId);
    return row.id;                                              // ← CHANGE THIS LINE
  } catch (err) {
    process.stderr.write(`[ObservationWriter] km-core putEntity (insight ${row.id}) failed: ${err.message}\n`);
    throw err;
  }
}
```
