|----------|-------------|
| HTTP handler → SnapshotManager.restoreSnapshot | snapshot id is user-controlled via URL path — validate before passing to git |

| SnapshotManager → host filesystem (.data/exports/) | git checkout writes files — already scoped via exportsRel + GIT_WORK_TREE |
| SnapshotManager → git binary | env var OKB_SNAPSHOT=1 crosses to child process — only meaningful inside the matched hook |
