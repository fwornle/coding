## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| stored context-turns / observation intents → DOM | preview, tool-arg, and observation-intent strings rendered in the modal are untrusted text |
| router param :taskId → fetch | fullscreen route reads :taskId; used only in a same-origin fetch path (no injection surface beyond URL encoding) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-86-03-01 | Information disclosure | secret leaking into a rendered turn preview / tool-arg digest | mitigate | Every preview/intent/arg string passes through the exported `scrubSecrets` before render (Task 1 grep-gate); raw bodies stay in the redacted channel. |
| T-86-03-02 | Tampering (XSS) | unescaped preview/tool text in the modal | mitigate | React default escaping; NO `dangerouslySetInnerHTML` (grep-gated == 0). All strings treated as untrusted text. |
| T-86-03-03 | Spoofing | experiment-cell narrative bleed (foreign-session intents rendered on a cell's turns) | mitigate | Preserve the `isExperimentCell` guard (grep-gated ≥2); new components align on per-cell-correct ContextTurnRow and do NOT re-run the observation join for cells (Pitfall 4). |
| T-86-03-04 | Information disclosure | fabricated cache_write=0 on OpenAI-wire turns in the band/modal | mitigate | ContextBand + modal branch on cache_write===null → CACHE_WRITE_NA (inherited from Plan 02). |
| T-86-03-SC | Tampering | npm installs | mitigate | No packages installed — Radix Dialog/react-router already present. No slopcheck surface. |
</threat_model>
