| Coding-only gate | `system === 'coding'` predicate enforced in component AND in route mount; defense in depth |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-55-11-01 | Cross-system data leak | OKB tab renders coding-only surfaces | mitigate | gate on `system === 'coding'` at BOTH render time (component) AND mount time (FilterRail / UnifiedViewer) |
| T-55-11-02 | XSS | hierarchy names / session IDs rendered | mitigate | React text-node escaping; entity names already validated server-side |
| T-55-11-03 | Information disclosure | LSL session metadata (start/end time, observation count) shown in tooltip | accept | localhost-only; same trust class as `/api/coding/observations` already exposed |
| T-55-11-04 | Performance | rapid window changes trigger many fetches | mitigate | TanStack Query caches per window key; debounce only if observed in production smoke |

Package legitimacy: no new packages.
