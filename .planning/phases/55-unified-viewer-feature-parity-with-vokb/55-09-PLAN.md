## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-55-09-01 | XSS | sub-tab content renders descriptionSegments / occurrence text from entity payload | mitigate | React text-node escaping by default; Evolution/Timeline content does NOT use `dangerouslySetInnerHTML` |
| T-55-09-02 | Reverse-tab-nabbing | Sources & Evidence external links | mitigate | `target="_blank" rel="noopener noreferrer"` on every external `<a>` |
| T-55-09-03 | Information disclosure | client confidence heuristic exposes scoring logic | accept | heuristic is identical to VOKB NodeDetails.tsx:165-213; not security-sensitive |
| T-55-09-04 | Denial of service | Confidence endpoint hit on every sub-tab click | mitigate | cache result by `selectedNodeId` in component state; only refetches when entity changes |

Package legitimacy: no new packages.
