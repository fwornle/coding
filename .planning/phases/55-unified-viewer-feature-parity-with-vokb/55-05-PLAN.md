<interfaces>
Extended StrokeStyle (additive on the existing Phase 45 shape):

```typescript
export interface StrokeStyle {
  width: number
  color: string
  opacity: number
  glow?: { color: string; size: number }
  borderStyle?: 'solid' | 'dashed'                              // NEW Phase 55
  halo?: { color: string; phase: number }                        // NEW Phase 55 (phase 0..1 of 1500ms cycle)
}

export function evaluatePulseRule(rule: string | null, entity: Entity): boolean
```

Color/shape fallback rules (UI-SPEC §14 — renderer applies in this order):
1. `display.color` || `classColor(className, theme)` (Phase 45 hash fallback)
2. `display.shape` || `shapeFallback(className)` || `'circle'`
3. `display.icon` || (no icon overlay)
4. `display.borderStyle === 'dashed'` OR `entity has 0 relations in current view` → dashed; else solid
5. `display.pulseRule && evaluatePulseRule(rule, entity)` → halo; else no halo

Pulse rule expressions (Phase 55 v1):
- `null` → no pulse
- `lastUpdatedWithin:60s` → `Date.now() - new Date(entity.updatedAt).getTime() < 60_000`
- `lastUpdatedWithin:5m` → same, 300_000
- `recentlyMerged:1h` → `entity.metadata.occurrences.some(o => Date.now() - new Date(o.timestamp).getTime() < 3_600_000)`

Reduced motion (UI-SPEC §12 + §15): when `window.matchMedia('(prefers-reduced-motion: reduce)').matches`, halo renders as static 50%-opacity ring (NO animation).
