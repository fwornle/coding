threat_model:
  - id: T-45-03-01
    threat: "XSS via entity.description rendering — a maliciously authored Markdown description executes `<script>` in the side panel."
    category: Tampering
    severity: high
    disposition: mitigate
    mitigation: "EntityDetailPanel uses the lightweight `markdown-text.tsx` (verbatim port from dashboard) — it does NOT use `dangerouslySetInnerHTML` and does NOT include `rehype-raw`. Plan 04 will introduce the full `react-markdown` panel; that one also rejects raw HTML by default. Add a Vitest in `EntityDetailPanel.test.tsx` that injects `<script>alert(1)</script>` and asserts it renders as escaped text, not as a DOM script."
  - id: T-45-03-02
    threat: "Substring search includes `description` field which may contain HTML or unicode control chars — slow regex / ReDoS."
    category: Denial of Service
    severity: low
    disposition: mitigate
    mitigation: "Use plain `String.prototype.includes()` (NOT regex). Lowercase both sides once via memoized derived selector. Description size cap acceptable since entities pre-truncated at ingest."
  - id: T-45-03-03
    threat: "Class multi-select payload arrives empty from a stale ontology fetch — search by class returns 'No matches' for valid graphs."
    category: Information Disclosure
    severity: low
    disposition: mitigate
    mitigation: "Empty `selectedClasses` Set is interpreted as 'all classes visible' (NOT 'no classes'). Test 2 in FilterRail asserts this branch."
  - id: T-45-03-04
    threat: "Keyboard shortcut `/` triggers inside an `<input>` and breaks user typing slashes."
    category: Denial of Service
    severity: medium
    disposition: mitigate
    mitigation: "useKeyboardShortcuts checks `document.activeElement?.tagName === 'INPUT' || 'TEXTAREA'` and skips when input has focus — except the global `Esc` which always fires. Test 3 in keyboard hook covers this."
  - id: T-45-03-05
    threat: "Accessibility regression — icon-only controls ship without aria-label, screen readers announce them as 'button'."
    category: Repudiation
    severity: medium
    disposition: mitigate
    mitigation: "`IconButton` primitive REQUIRES `ariaLabel: string` (TS-narrowed; non-optional). `IconButton.a11y.test.tsx` uses testing-library `getByRole('button', {name})` to assert each control's accessible name matches the UI-SPEC § Icon-only controls table verbatim."
