# CodeKnowledgeGraphModule

**Type:** SubComponent

## Observations

- CodeKnowledgeGraphModule constructs and manages the code knowledge graph
- The CodeKnowledgeGraphModule relies on the GraphDatabaseModule for graph database adaptation and persistence
- CodeKnowledgeGraphModule utilizes the EntityPersistenceModule for entity storage and relationship management
- CodeKnowledgeGraphModule generates the code knowledge graph for both ManualLearning and OnlineLearning
- The CodeKnowledgeGraphModule utilizes the PersistenceAgent (integrations/mcp-server-semantic-analysis/src/agents/persistence-agent.ts) for entity storage and relationship management
- CodeKnowledgeGraphModule provides a comprehensive representation of the codebase
- CodeKnowledgeGraphModule enables semantic code search and analysis
